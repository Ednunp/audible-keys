# Audible Keys

Simple keyboard shortcuts for the Audible web player — play/pause, skip, volume, speed, chapters and bookmarks. Built to be screen-reader friendly, and handy for everyone.

## Keys (while the Audible player tab is open)
- **P** — play / pause
- **Left / Right** — skip back / forward 30 seconds
- **Up / Down** — volume
- **Page Up / Page Down** — previous / next chapter
- **S / Shift+S** — speed up / down · **Home** — back to normal speed
- **B** — add a bookmark · **Shift+B** — open bookmarks list · **C** — open chapters

It also remembers the speed you set for each book, and a few of the keys can be set to work even when Chrome isn't the focused window. The extension's built-in Help page has the full details and setup.
